Music API

A simple API for managing music albums, artists, and concerts.

Albums

GET /albums - List all albums
GET /albums/:id - Get album details
POST /albums - Create new album
DELETE /albums/:id - Delete album

Artists

GET /artists - List all artists
POST /artists - Add new artist
PUT /artists/:id - Update artist

Concerts

GET /concerts?minDate=YYYY-MM-DD&maxDate=YYYY-MM-DD - List all concerts within a date or time range
PATCH /concerts/:id - Update concert date or time

Tracks

GET /albums/:id/tracks - List album tracks
POST /albums/:id/tracks - Add track to album
DELETE /albums/:albumId/tracks/:trackNumber - Remove track
